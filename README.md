# Elegant Nails 💅✨

Elegant Nails es una One Page Website moderna y elegante diseñada para un salón de belleza y estética de uñas. El sitio condensa toda la información esencial de forma fluida, permitiendo a las clientas explorar servicios, ver trabajos reales y reservar un turno directamente desde un dispositivo móvil o computadora.

## 🚀 Características

- **Diseño One Page:** Navegación interna dinámica por secciones sin recargar la página.
- **Totalmente Responsive:** Optimizado con Media Queries para ofrecer una experiencia perfecta en celulares, tablets y computadoras.
- **Llamados a la Acción (CTA):** Botón de reserva enlazado directamente a WhatsApp con un mensaje predefinido.
- **Estética Cuidada:** Paleta de colores verde salvia y tipografías que transmiten sofisticación y profesionalismo.

## 🛠️ Tecnologías utilizadas

- **HTML5:** Estructura semántica del sitio.
- **CSS3:** Estilos personalizados, diseño de grillas (Grid), flexbox y animaciones de transición.
